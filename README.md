# Vizsgaremek – Backend
Jedlik 14KE IT eszköz nyilvántartó backend repository

- **Csapattagok:** Lukács Gábor, Takács Gábor, Tóth Krisztián

## Leírás
Ez a repository a vizsgaremek projekt backend részét tartalmazza.  
A backend REST API-t biztosít a frontend számára, és MySQL adatbázissal kommunikál.

## Használt technológiák
- ASP.NET Core (.NET 8)
- Entity Framework Core
- MySQL / MariaDB
- REST API
- JSON alapú kommunikáció
